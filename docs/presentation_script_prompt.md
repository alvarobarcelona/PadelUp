# PadelUp - Video Presentation Guide

This document contains everything needed to generate a short (30-40s) presentation video for **PadelUp**, including key features, a script, and AI prompts.

## 1. App Overview & Key Features

**PadelUp** is a comprehensive Padel companion app designed to track progress, organize matches, and connect players.

### Core Features to Highlight

- **Dynamic Dashboard**: Visual display of current Level, ELO rating, and progress bar to the next level.
- **Easy Match Recording**: "New Match" feature to select players, input scores (sets), and automatically calculate new ELO ratings based on match results.
- **Smart Analytics**: Match history, win/loss tracking, and performance stats.
- **Social & Community**: Player suggestions (based on ELO), chat functionality, and leaderboards.
- **Tournaments & Events**: Integration with tournament platforms.

## 3. AI Prompts for Video Generation

### Option A: Complete Video Direction (for Video AI)

> **Prompt:** "Create a dynamic 30-second commercial for a mobile app called 'PadelUp'. The visual style should be modern, sleek, using a dark mode UI with vibrant green and blue accents.
